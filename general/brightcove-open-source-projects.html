Proyectos de código abierto de Brightcove

A continuación se muestra una lista de todos los proyectos de código abierto actuales para las tecnologías de Brightcove, agrupados por categoría.
Brightcove no admite oficialmente los proyectos de código abierto.

Bienvenido a Brightcove Open Source. Aquí encontrará varios proyectos creados por empleados de Brightcove o miembros de la comunidad de Brightcove para ayudar a acelerar el desarrollo de Brightcove.

Si está interesado en contribuir a proyectos, vaya a los sitios de proyectos individuales en Github. Si tiene un proyecto que le gustaría enumerar aquí, Contáctenos.

API actuales

  1. Script de resolución de problemas de solicitud de API

    Solucione problemas de solicitudes y respuestas HTTP hacia / desde las API de Brightcove.

    https://github.com/BrightcoveOS/api-request-troubleshooting-script

  2. Cargador de Brightcove

    Módulo frontend para carga directa de varias partes a s3 con salsa secreta para la ingesta de Brightcove.

    https://github.com/BrightcoveOS/evaporate-brightcove

  3. Contenedor de API de reproducción de Brightcove

    Este es un contenedor de PHP para API de reproducción de Brightcove escrito por Theresa Newman.

    https://github.com/BrightcoveOS/Brightcove-Playback-API-Wrapper

  4. Envoltorio PHP para Brightcove Dynamic Ingest

    Este proyecto proporciona un punto de partida para la integración de Brightcove API de ingesta dinámica en su aplicación. Proporciona formas sencillas de agregar y actualizar videos y medios asociados mediante la ingesta basada en extracción o la carga de archivos de origen.

    https://github.com/BrightcoveOS/php-wrapper-dynamic-ingest

  5. Fachada API

    Esta humilde biblioteca pretende ser una fachada simple sobre las API de servidor de Brightcove. Como la mafia exige funcionalidad adicional, se agregará aquí.

    https://github.com/nwbb/node-brightcove

  6. Cliente API de gestión de jugadores

    Este módulo se puede utilizar para acceder a la API de administración de reproductores de Brightcove. Se puede encontrar documentación para la API subyacente aquí.

    https://github.com/BrightcoveLearning/Player-Management-Client

  7. Ejemplos de API de Brightcove con CSharp

    Este proyecto muestra cómo se pueden utilizar las nuevas API de Brightcove con C #.

    Tenga en cuenta que esto funcionará desde entornos de escritorio (probado), pero probablemente no funcionará si se usa en páginas ASP.NET debido a CORS. Si se encuentra con este problema, creo que la única forma de resolverlo es utilizar el proxy PHP proporcionado por Brightcove en todos sus API de CMS páginas de ejemplo.

    Todo el código se proporciona tal cual y únicamente como ejemplo. Úselo bajo su propio riesgo. No se le proporciona soporte de ninguna manera, forma o forma.

    https://github.com/houseofmackee/BrightcoveCS

  8. Herramientas de Python de las API de Brightcove

    Este repositorio contiene herramientas y módulos escritos en Python para manejar las API relacionadas con Brightcove y simplificar tareas como iterar sobre la biblioteca de videos completa y realizar acciones en los videos.

    Versiones probadas de Python: 3.8.3

    Todo el código se proporciona tal cual y únicamente como ejemplo. Úselo bajo su propio riesgo. No se le proporciona soporte de ninguna manera, forma o forma.

    https://github.com/houseofmackee/BrightcovePY

  9. Cliente API de gestión de jugadores

    Este módulo se puede utilizar para acceder a la API de administración de reproductores de Brightcove. Se puede encontrar documentación para la API subyacente aquí.

    https://github.com/BrightcoveLearning/Player-Management-Client

Reproductor y SDK

  1. Generador de marco de complementos de Video.js

    Este es un generador Yeoman obstinado para complementos de video.js. Se basa en las recomendaciones del equipo central de video.js, así como en las herramientas y convenciones para desarrollar complementos en Brightcove.

    https://github.com/videojs/generator-videojs-plugin

Integraciones de sistema

  1. Brightcove Video Connect para SharePoint 2013

    Brightcove Video Connect para SharePoint 2013 permite a los editores agregar, explorar y administrar videos y listas de reproducción de videos directamente desde SharePoint. Su componente de reproductor permite una fácil integración de los activos de Brightcove Video Cloud directamente en las páginas de SharePoint.

    http://go.brightcove.com/bc-video-connect-sharepoint2013

  2. Brightcove Video Connect para Drupal

    Brightcove Video Cloud es una solución de publicación y alojamiento de videos para ofrecer videos de calidad profesional a audiencias digitales en cualquier dispositivo. Ofrece reproductores altamente personalizables, streaming HD, sincronización con CDN de terceros y YouTube. Con Brightcove Video Connect para Drupal, puede agregar y administrar contenido de video de manera eficiente en sus sitios web con tecnología Drupal.

    https://www.drupal.org/project/brightcove

  3. Brightcove Video Connect para WordPress

    ¿Está buscando manejar su biblioteca Brightcove Video y Playlist de forma nativa desde WordPress? Con este complemento, desarrollado por 10up.com, tiene el poder de manejar múltiples cuentas y bibliotecas de videos, cargar videos y agregarlos a listas de reproducción, renderizar códigos cortos con sus videos y todo desde la interfaz de administración de WordPress.

    https://wordpress.org/plugins/brightcove-video-connect/

  4. Conector de SharePoint 2013

    Proporciona integración entre SharePoint 2013 y varias cuentas de Video Cloud.

    https://github.com/brightcove/sharepoint-connector

  5. Brightcove Video Connect para Adobe Experience Manager

    Brightcove Video Connect para Adobe Experience Manager (AEM) es la forma más sencilla de integrar el flujo de trabajo entre Brightcove Video Cloud y AEM. Con Brightcove Video Connect, los usuarios de AEM pueden agregar, explorar y administrar videos directamente desde la interfaz de usuario de AEM.

    http://go.brightcove.com/bc-video-connect-aem

  6. Componente CQ5

    El componente CQ5 integra Brightcove

    Tecnologías: CQ5

    https://www.github.com/coresecure/Adobe-CQ5-Brightcove-Connector

  7. montacargas

    Salesforce proporciona Bulk API para facilitar la sincronización masiva de datos desde un almacén de datos a Salesforce. Este proyecto proporciona una joya para usar Salesforce Bulk API en una aplicación ruby.

    Tecnologías: Salesforce, Ruby

    https://www.github.com/brightcove/salesforklift

Analítica

  1. Adobe Analytics

    El complemento de Adobe Analytics se puede configurar fácilmente mediante el módulo Reproductores. Para obtener información, consulte Configuración del seguimiento de eventos de Adobe Analytics.

  2. Google analitico

    El complemento de Google Analytics se puede configurar fácilmente mediante el módulo de reproductores. Para obtener información, consulte Configuración del seguimiento de eventos de Google Analytics.

    Esta es una aplicación de video de LG que muestra varias listas de reproducción de una cuenta de Brightcove. La aplicación ha sido diseñada para que sea necesaria una configuración mínima y permite a los clientes realizar personalizaciones para cambiar la interfaz de usuario o agregar / eliminar funciones de acuerdo con la funcionalidad que deseen.

    Tecnologías: LG, TV

    https://www.github.com/BrightcoveOS/LG-Sample-App

    Tecnologías: Roku, TV

  5. Java Commons

  6. Diamante

    Diamond es un demonio de Python que recopila métricas del sistema y las publica en Graphite. Es capaz de recopilar métricas de CPU, memoria, red, E / S, carga y disco. Además, cuenta con una API para implementar recopiladores personalizados para recopilar métricas de casi cualquier fuente.

    Tecnologías: Grafito, Python

    https://github.com/python-diamond/Diamond
