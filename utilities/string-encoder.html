Codificador de cadena

Utilidad para codificar cadenas de URI de 3 formas.

Agregue los nombres de métodos que sugieren, estos generalmente son útiles cuando necesita incluir alguna cadena en una URL. Cuál es el apropiado depende de la situación. A continuación se muestran algunos ejemplos de cómo los encontramos útiles.

  • escape es el método más antiguo y el que menos hace; no hemos tenido ocasión de usarlo en nuestras aplicaciones
  • encodeURI generalmente es adecuado para cadenas que se pasarán como valores para parámetros de URL
  • encodeURIComponent es el codificador más completo, pero generalmente solo lo necesitas para cadenas que se usarán como parte de un dominio/ruta de URL, o si necesitas pasar una URL como parámetro (lo hacemos con frecuencia al pasar solicitudes API a proxies a través de AJAX)
