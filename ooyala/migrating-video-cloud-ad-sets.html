    Migración a Video Cloud: Conjuntos de anuncios

    En este tema se explica cómo se migran los conjuntos de anuncios a Video Cloud.

    Nota: este tema sólo se aplica a los clientes de Ooyala que se migran a Video Cloud.

    Cómo se migran los conjuntos de anuncios

    Video Cloud no tiene el concepto de conjunto de anuncios. Los anuncios del lado del cliente se configuran generalmente a través de puntos de referencia de vídeo y un complemento publicitario para el reproductor. Consulte paso a paso: Implementación de Publicidad para obtener más detalles.

    Al migrar vídeos a Video Cloud asociados con un conjunto de as, estamos haciendo dos cosas:

    • Migrar algunos datos de conjuntos de anuncios a puntos de referencia (lo que se migra varía según el ad_source, y se detalla en las secciones siguientes)
    • Almacenar el identificador del conjunto de anuncios en un campo personalizado denominado ad_set_id, para que el vídeo pueda actualizarse más adelante para ver cambios en el conjunto de anuncios

    Si el vídeo se reproduce en un reproductor Ooyala existente, los anuncios deben seguir reproduciéndose como antes. En el caso de los reproductores de Brightcove, deberá agregar y configurar el plugin de anuncios apropiado al reproductor. Consulte Configuración de la publicidad del jugador mediante el módulo de reproductores y la otra documentación de publicidad para Brightcove Player para obtener más detalles.

    Fuentes de anuncios admitidos

    Solo se admiten los siguientes ad_source tipos para migrar datos de conjuntos de anuncios a Video Cloud:

    • Videoplaza
    • Google IMA v3
    • FreeWheel
    • Mis anuncios
    • VAST
    • VPAID

    Los siguientes campos se eliminarán para todos los orígenes de la lista anterior que lo admitan:

    • tracking_pixel_url
    • ad_frequency

    Los campos adicionales para estos ad_sources se almacenarán como cue_points[i].metadata del tipo «CODE» como JSON stringified. Consulte las secciones siguientes para obtener asignaciones exactas.

    Videoplaza

    Asignaciones de datos de Videoplaza
    Datos del conjunto de anuncios de Ooyala Datos de vídeo de Video Cloud
    ad_source [=videoplaze] cue_points[i].name
    ad_position cue_points[i].time
    vpDomain cue_points[i].metadata.vpd
    player_level_midroll_ad_breaks cue_points[i].metadata.pmab
    player_level_tags cue_points[i].metadata.pt
    player_level_videoplaza_flags cue_points[i].metadata.pvp
    player_level_shares cue_points[i].metadata.ps
    short_form_content_max_length cue_points[i].metadata.scml
    non_linear_ad_breaks cue_points[i].metadata.nlab
    account_id cue_points[i].metadata.account_id

    Asignaciones de datos de Google IMA v3

    ad_position cue_points[i].time
    ad_tag cue_points[i].metadata.ad_tag
