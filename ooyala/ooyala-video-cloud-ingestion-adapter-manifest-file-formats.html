Ooyala-Video Cloud: Formatos de archivo de manifiesto

Este tema cubre los formatos de archivo de manifiesto que puede usar para ingerir videos y activos.
Nota: este tema se aplica solamente a los clientes de Ooyala migrados a Video Cloud.

Introducción

Al ingerir videos, puede usar archivos de manifiesto para especificar metadatos adicionales (incluido el título del video, la descripción, los tiempos de vuelo, las etiquetas y los metadatos personalizados), así como los archivos de miniaturas y subtítulos opcionales. Los archivos de manifiesto están en formato XML o CSV. Para obtener detalles sobre los manifiestos, consulte:

  • Archivos de manifiesto CSV
  • Archivos de manifiesto XML

Cuando comienza el procesamiento

Nota: Este tema se aplica a OIS v2.5.
Nota: Video Cloud no comienza a procesar ninguno de los videos hasta que se recibe el archivo de manifiesto XML o CSV. Como resultado, es importante elegir una estrategia de carga por lotes. Si sus videos deben estar disponibles rápidamente (por ejemplo, distribución de noticias), debe crear lotes muy pequeños o incluso por lotes individualmente (un archivo CSV o XML por video).
