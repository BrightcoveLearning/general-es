Introducción

La nube de video API de análisis es bastante similar a la API REST de Ooyala Analytics.

Tenga en cuenta también que el módulo de análisis en Video Cloud Studio incluye un sólido Generador de informes personalizados que en muchos casos puede hacer que no sea necesario utilizar la API.

Las secciones siguientes corresponden a los diferentes aspectos de las dos API:

  • Autenticación
  • Informes
  • Parámetros
  • Dimensiones
  • Filtros
  • Métrica

Autenticación

La autenticación de las solicitudes de la API de Brightcove se maneja de forma ligeramente diferente a la de las API REST de Ooyala. Ver Autenticación para solicitudes de API de Brightcove para detalles.

Informes

El soltero /reports El punto final para la API REST de Ooyala Analytics corresponde muy de cerca a Brightcove API de análisis /data punto final. La principal diferencia es que la API de Ooyala permite ambos GET y POST, esta última solía pasar algunos datos al cuerpo de la solicitud en lugar de parámetros URL, lo que permite cadenas de datos más largas.

Dimensiones

Las API de Ooyala y Brightcove Analytics agrupan datos en depósitos llamados "dimensiones". Se usan de manera similar, pero las API tienen reglas algo diferentes en cuanto a cómo se pueden combinar múltiples dimensiones, y Brightcove tiene un conjunto más amplio de dimensiones. Ver Combinaciones de dimensiones admitidas para obtener detalles sobre la API de Brightcove. La siguiente tabla proporciona una comparación detallada.

Filtros

Los filtros funcionan de manera prácticamente idéntica en las dos API, especificadas en términos de:

      
        // Ooyala
        filter={dimension1}=={value1};{dimension2}=={value2}
      
        // Brightcove
        where={dimension1}=={value1};{dimension2}=={value2}

Métrica

La metrics(para la API de Brightcove: fields) devuelta en los informes varía según las dimensiones solicitadas para la API de Brightcove Analytics. Puede utilizar la herramienta incorporada en Dimensiones y campos para ver qué campos se pueden devolver para una dimensión o combinación de dimensiones en particular.

Tenga en cuenta que, si bien la API de Ooyala devuelve todas las métricas si no se especifica ninguna, la API de Brightcove solo devuelve un pequeño conjunto de métricas predeterminadas que varían según la dimensión (combinación) a menos que incluya la fields en la solicitud.

Los campos de Brightcove contienen equivalentes para todas las métricas de Ooyala excepto:

  • uniq_plays_requested
  • uniq_displays
  • uniq_video_starts

El Brightcove API de análisis también puede devolver un número de otras métricas no disponibles en el análisis de Ooyala.
