Ooyala-Video Cloud: Archivos de manifiesto CSV

Puede continuar usando FTP o Aspera con un manifiesto CSV para transferir sus videos a Video Cloud. Este tema explica la estructura del CSV.
Nota: este tema se aplica solamente a los clientes de Ooyala migrados a Video Cloud.

Introducción

Al ingerir videos, puede usar archivos de manifiesto CSV para especificar metadatos adicionales (incluido el título del video, la descripción, los tiempos de vuelo, las etiquetas y los metadatos personalizados), así como los archivos de miniaturas y subtítulos opcionales. Puede utilizar archivos de manifiesto CSV cuando ingiera contenido con FTP o Aspera.

Limitaciones

Nota: Este tema se aplica a OIS v2.5.

El formato CSV del archivo de metadatos admite menos funciones que el formato XML. El formato CSV le permite especificar miniaturas (imágenes de vista previa) y metadatos personalizados para asociarlos con un video. El formato XML admite estas características, así como la asociación de los nombres de conjuntos de anuncios definidos existentes.

Manejo de metadatos

Si incluye metadata para los videos cuando los ingiere, se asigna a los campos de video de Video Cloud de una de estas dos formas:

  1. De forma predeterminada, cada par metadata clave/valor se añade como etiqueta en la forma: meta::key::value. Por ejemplo, <ooyala:metadata name="category">cooking</ooyala:metadata> produciría la etiqueta meta::category::cooking
  2. Si prefieres usar custom_fields para almacenar metadatos, Brightcove Support puede cambiar una configuración en su cuenta para hacerlo. Tenga en cuenta lo siguiente:
    • Deberá crear campos personalizados con nombres que correspondan a sus claves de metadatos.
    • El número de campos personalizados está limitado a 10 de forma predeterminada, por lo que si necesita más para acomodar sus metadatos, deberá solicitar campos personalizados adicionales al Soporte.
    • Si un campo personalizado correspondiente a cada elemento de metadatos no existe en su cuenta, la operación de metadatos completa fallará.

Encabezados de columna de manifiesto CSV

Los siguientes campos son la primera línea (encabezados de columna) de su archivo CSV, con los campos de datos reales a continuación en las filas subsiguientes (una fila por video).

Nota: Puede especificar estas columnas en cualquier orden y puede omitir las columnas que no necesita, siempre que los datos de las filas de datos coincidan exactamente con los encabezados.

Creación de un archivo de manifiesto CSV

Tenga en cuenta lo siguiente al generar un archivo de metadatos CSV:

  • Si un valor de texto en un campo de datos tiene una coma (,), debe encerrar el campo entre comillas dobles (").
  • Si un valor de texto en un campo de datos tiene una comilla doble («) en él, debe incluir el campo entre comillas dobles y escapar del carácter de comillas dobles en el campo de texto con una segunda comilla (por ejemplo, «Ella dijo: «» Que tenga un buen día. «"»)

Si está creando su archivo de metadatos CSV desde Excel, automáticamente maneja el escape.

Para crear un archivo de metadatos CSV:

  1. Sube uno o más videos.
  2. Abra un programa de hoja de cálculo, como Microsoft Excel.
  3. Agregue los encabezados de las columnas a la primera fila.
  4. Agrega una fila para cada video.
  5. Guarde el archivo como CSV. Por ejemplo, puede seleccionar Guardar como desde el Archivo menú. Cuando se le solicite, seleccione el tipo de archivo de valores separados por comas.
  6. Con el método de carga que elija, cargue el archivo CSV a medida que subió los videos. Ver cualquiera FTP o Aspera.
